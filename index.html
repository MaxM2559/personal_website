Interactive Cluster Map with Automatic Categories

A mapping tool that creates a map and marks every point of data from a LatLong data set. These points are automatically sorted by their given category variable and can be manually hidden/clustered with the layer control menu at the top right. The focus for this project was a proof of concept that a map representing thousands of data points while being user readible was possible and doesn't demand much processing. Demo

TED Finds: A Search Engine to Find Relavent Ted Talks

This web-based tool recommends TED Talks similar to a user-selected talk by analyzing their content and topics. It also provides a breakdown of the depth of coverage for various subjects within each talk. The project utilizes fundamental natural language processing techniques, including edit distance, cosine similarity, singular value decomposition (SVD), TF-IDF, and sentiment analysis, to deliver meaningful recommendations.

Creating the first International Industry Code Corrospondence Sheet

TThis project developed a suite of tools to generate comprehensive correspondence sheets for international industry codes, including NAICS, HS, ISIC, CPC, and their variations. Unlike existing resources that only mapped codes across years or between two systems, these tools create unified correspondence sheets capable of translating across all publicly available variations. GitHub Page.

Research Project: Participation of team sports by high school students influence on underage drinking?

As part of a class of a research team, I analyzed the impact of high school students' regular participation in team sports on their likelihood of engaging in risky behaviors such as gambling. Using data from the 2006 High School Risk Survey, we used Stata to conduct linear , multilinear , and logistic regression analyses. Our findings revealed a positive and statistically significant relationship between sports participation and gambling-like behavior. Final Report

Lambda Ledge: Game Level Generator

As part of the semester long final project for the CS3110 Functional Programming course, my team developed a working 2D platformer game. Our main focus was creating an intuitive level generator, individuals with no coding experience could easily create their own level designs in a text file which is then read and turned into an actual level in the game. Lambda Ledge is complete with movement, powerups, and levels. GitHub page.
